# Keypad Controlled Robot Car
This project demonstrates a Keypad Controlled Robot Car using Arduino and a 4x4 matrix keypad. The robot is controlled by pressing keys on the keypad, which sends input signals to the Arduino to control the robot’s movement.

# Features
* Direct hardware control
* Simple and reliable system
* No wireless communication required
* Easy to understand for beginners

# Components Used
* Arduino Uno / Arduino Nano
* 4x4 Matrix Keypad
* L298N Motor Driver
* DC Motors
* Robot Chassis
* Battery Pack
* Breadboard
* Jumper Wires

# Working Principle
The keypad acts as an input device. When a key is pressed, the Arduino detects the signal and processes it according to the programmed logic. The Arduino then sends signals to the motor driver to control the robot’s movement.


# Applications
* Robotics learning
* Embedded systems experiments
* Educational projects
* Basic automation systems
