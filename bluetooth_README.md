# Bluetooth Controlled Robot Car 
This project demonstrates a Bluetooth Controlled Robot Car using Arduino. The robot can be controlled wirelessly using a smartphone through a Bluetooth module. It allows users to send commands such as forward, backward, left, and right using a mobile application.

# Features
* Wireless control using smartphone
* Real-time robot movement
* Simple and cost-effective design
* Easy integration with Arduino

# Components Used
* Arduino Uno / Arduino Nano
* HC-05 or HC-06 Bluetooth Module
* L298N Motor Driver
* DC Motors
* Robot Chassis
* Battery Pack
* Jumper Wires

# Working Principle
The Bluetooth module connected to the Arduino receives commands from a smartphone via Bluetooth communication. These commands are transmitted through serial communication to the Arduino, which processes them and controls the motor driver to move the robot accordingly.

# Applications
* Wireless robotics projects
* Remote vehicle control
* Educational robotics learning
* Home automation experiments
